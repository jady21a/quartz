## 步骤
### 1.部署网站到公网
- GitHub Pages
- Netlify
- Vercel
- Cloudflare Pages
### 2. **提交到搜索引擎

#### **Google Search Console**(推荐优先)

1. 访问 [Google Search Console](https://search.google.com/search-console)
2. 添加你的网站域名
![[Pasted image 20251120100947.png]]
3. 验证网站所有权(通常通过添加 HTML 文件或 DNS 记录)
4. 提交 Sitemap:`https://你的域名/sitemap.xml`
![[Pasted image 20251120125822.png]]
5. 请求编入索引
#### **Bing Webmaster Tools**

1. 访问 [Bing Webmaster](https://www.bing.com/webmasters)
2. 添加网站并验证
3. 提交 Sitemap

#### **其他搜索引擎**

- 百度站长平台(如果目标用户在中国)
- Yandex Webmaster(俄罗斯用户)
### 3. **优化 Quartz 配置**
在 `quartz.config.ts` 中确保启用 SEO 相关功能:
```ts
plugins: {
  transformers: [
    // ... 其他插件
    Plugin.Description(), // 生成页面描述
  ],
  emitters: [
    Plugin.ComponentResources(),
    Plugin.ContentPage(),
    Plugin.FolderPage(),
    Plugin.TagPage(),
    Plugin.ContentIndex(),
    Plugin.Assets(),
    Plugin.Static(),
    Plugin.NotFoundPage(),
    Plugin.Sitemap(), // 确保启用 Sitemap
  ],
}
```

### 4. **创建 robots.txt**
在 `quartz/static/` 目录下创建 `robots.txt`:
```txt
User-agent: * 
Allow: / 
Sitemap: https://你的域名/sitemap.xml
```
确保 sitemap.xml 存在
访问https://你的域名/sitemap.xml,能看到许多 `<url>` 标签就 OK。

### 5. 检查是否收录
在搜索框输入：site:你的域名
预期时间
- Google:通常 **1-4周** 开始收录
- Bing:通常 **2-6周**
- 百度:可能需要 **1-3个月**


### 5. 优化
#### 优化内容以提高收录
- 添加有意义的标题和描述
- 在笔记中使用关键词
- 添加内部链接
- 保持内容更新
- 确保页面加载速度快
#### **获取外部链接**
- 在社交媒体分享你的网站
- 在相关论坛或社区发布链接
- 与其他网站交换友情链接
